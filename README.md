# 🏥 Hospital Emergency Room Dashboard | Excel

This project is an interactive **Hospital Emergency Room Dashboard** built in Microsoft Excel to analyze patient flow, admission status, waiting time, and departmental referrals. The dashboard provides a
comprehensive view of emergency room operations and helps monitor key performance indicators for better healthcare management.

The report includes dynamic filters for **Year and Month**, enabling users to explore trends and compare patient statistics over different periods.

## Key Features
- KPI cards displaying **Total Number of Patients, Average Wait Time, and Patient Satisfaction Score**.
- Analysis of patients attended on time versus delayed cases.
- Gender-wise patient distribution.
- Admission status analysis showing admitted and non-admitted patients.
- Age group-wise patient segmentation.
- Department referral analysis across specialties such as General Practice, Orthopedics, Cardiology, Neurology, and Physiotherapy.
- Interactive monthly and yearly filtering for detailed analysis.

## Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Visualization

## Insights Generated
- Evaluated emergency room efficiency through wait time and patient satisfaction metrics.
- Identified patient distribution by gender and age groups.
- Analyzed admission rates and treatment delays.
- Examined department-wise referrals to understand patient demand patterns.

This project demonstrates proficiency in **Excel dashboard development, data analysis, and data visualization**, showcasing the ability to transform healthcare data into meaningful insights for operational and
decision-making purposes.
